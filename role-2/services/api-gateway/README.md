# API Gateway

Role 2 Go API gateway boilerplate.

Public endpoints are limited to:

- `GET /health`
- `POST /api/troubleshoot`
- `POST /api/troubleshoot/unresolved`
- `POST /api/incidents/escalate`
- `GET /api/incidents`
- `GET /api/incidents/{incident_id}`
- `GET /api/machines`
- `GET /api/telemetry/summary`
